Project Name: Personal Finance Analytics Tool

Description: Developed a Python-based application to automate the tracking and analysis of daily financial transactions.Leveraged Pandas for efficient data manipulation and cleaning of transaction records stored in CSV format.Implemented data visualization using Matplotlib to generate categorical spending insights and monthly trends.
Designed a user-friendly command-line interface (CLI) for seamless data entry and persistence.

Key Learning: File I/O, Data Cleaning, aur Data Visualization.
